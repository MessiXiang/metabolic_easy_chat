workflow

从想法到变更，路径更短

  1. 连接模型配置兼容 OpenAI 的服务与默认模型。
  2. 打开工作区把代码文件、终端和对话放在同一上下文。
  3. 生成与执行流式回复、图片生成、补丁应用和命令执行一起完成。

metabolism

新陈代谢流程

让应用跟着代码库一起进化：从本地对话修改，到 PR 审核，再回到 App 内更新。

01

点击新陈代谢

在 App 内点击“新陈代谢”按钮，启动代码库更新流程。

02

配置 GH 工具

如需提交代码或创建 PR，先按提示配置 GitHub 相关工具与权限。

03

与 AI 对话改代码

描述想要的功能或修复，AI 会结合工作区上下文更新代码库。

04

提交并 PR

确认变更后点击“提交并 PR”，把修改提交给原仓库维护者审核。

05

作者核实合并

原作者检查代码、确认安全性和质量，通过后 merge 到主分支。

06

更新最新内容

回到 App，点击左上角更新按钮，拉取最新代码与应用内容。

install

下载安装

下载最新版本
  1. 下载 zip前往 GitHub Releases，下载最新的 metabolic_easy_chat.zip。
  2. 解压文件双击 zip 解压，得到 metabolic_easy_chat.app。
  3. 拖到应用程序把 App 拖进 macOS 的“应用程序”文件夹。
  4. 打开 App在“应用程序”里打开 metabolic_easy_chat。
  5. 如果被拦截到“系统设置” → “隐私与安全性”，找到被阻止的应用，点击“仍要打开”或“允许打开”。

提示：首次打开时 macOS 可能会进行安全检查。如果普通双击打不开，可以先右键 App，然后选择“打开”。
